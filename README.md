# GMV Chuvas - Fazenda Matão

Projeto limpo v1.0.

## Como publicar no GitHub Pages
1. Apague os arquivos antigos do repositório.
2. Envie todos os arquivos e pastas deste pacote: `index.html`, `css/`, `js/`, `assets/`, `kml/`, `manifest.json`, `icone.svg`, `README.md`.
3. Aguarde 1 a 2 minutos.
4. Abra o link do GitHub Pages e pressione Ctrl+F5 no computador.

## Importante
Os polígonos dos talhões estão embutidos no arquivo `js/talhoes.js`, gerados a partir do KML real `FMatao2526.kml`.
